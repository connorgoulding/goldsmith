# Goldsmith Growth — Nature is the Algorithm

A biohacking-inspired landing page built with React, GSAP, and Tailwind CSS.

## Features

- **Floating Island Navbar** — Glassmorphism navigation with smooth hover effects
- **Immersive Hero** — Full-screen forest imagery with gradient overlays
- **Micro-UI Cards** — Diagnostic shuffler, telemetry typewriter, protocol scheduler
- **Sticky Protocol Stacking** — Scroll-triggered card animations
- **Philosophy Manifesto** — Bold typographic contrast
- **Noise Texture Overlay** — Subtle grain for organic feel

## Tech Stack

- React 18
- GSAP + ScrollTrigger
- Tailwind CSS
- Vite
- Plus Jakarta Sans (Google Fonts)

## Getting Started

```bash
npm install
npm run dev
```

## Build for Production

```bash
npm run build
```

## Deployment

Built files are in `dist/` — deploy to Netlify, Vercel, or Cloudflare Pages.
